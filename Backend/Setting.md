### Server 구동

- 가상환경 활성화

  ```
  source venv/Scripts/activate
  ```

  

- 라이브러리 설치

  ```
  # bert 관련 - 처음에만 필요
  pip install mxnet
  pip install git+https://git@github.com/SKTBrain/KoBERT.git@master
  ```

  ```
  pip install -r requirements.txt
  ```



- migration

  ```
  python manage.py migrate
  ```

  

- 실행

  ```
  python manage.py runserver
  ```
